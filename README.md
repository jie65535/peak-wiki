## 关于PEAK

**PEAK**是一款由Team PEAK开发、Aggro Crab和Landfall发行的多人合作攀登游戏，于2025年6月16日在Steam平台发布。

- **开发商**：Team PEAK
- **发行商**：Aggro Crab, Landfall
- **平台**：Steam (Windows, SteamOS)
- **类型**：多人合作
- **模式**：单人或多人（最多4人）
- **引擎**：Unity
- **作曲**：Erik Skog
- **Steam页面**：[https://store.steampowered.com/app/3527290/PEAK/](https://store.steampowered.com/app/3527290/PEAK/)

### 游戏简介

玩家扮演童子军（Scout），从BingBong航空公司的飞机坠毁后醒来，目标是到达岛屿最高点——顶峰（Peak），并使用信号弹召唤救援直升机。游戏支持单人或最多4人合作，设有多个难度等级（从新手到攀登7）。

岛屿的地形和生物群系每天UTC时间17:00随机生成，每次探险都是全新的挑战！
