# 贡献指南

感谢您对PEAK中文Wiki的贡献兴趣！

## 如何贡献

### 内容贡献

1. **确保准确性**
   - 所有信息必须基于游戏实际内容
   - 如有不确定，请在游戏中验证
   - 引用英文Wiki时请确保翻译准确

2. **使用中文**
   - Wiki内容使用简体中文编写
   - 游戏术语首次出现时附上英文原文，如：侦察员（Scout）
   - 保持术语翻译一致性

3. **遵循格式**
   - 查看现有文档的格式和结构
   - 使用Markdown语法
   - 保持文档结构清晰

4. **添加图片**
   - 图片存放在 `docs/images/` 相关目录
   - 道具图标：x512和x64两种规格
   - 截图：使用游戏内截图

### 提交流程

1. Fork本仓库
2. 创建新分支
3. 进行修改和添加
4. 提交Pull Request
5. 等待审核

### 内容优先级

当前需要的内容（按优先级）：
1. 道具详细信息和图标
2. 生物群系详细描述
3. 游戏机制说明
4. 游戏指南和攻略
5. 截图和演示

## 编辑规范

### Markdown格式

- 标题使用 `#` 到 `####`
- 列表使用 `-` 或 `1.`
- 代码块使用三个反引号
- 链接格式：`[文字](链接)`
- 图片格式：`![描述](路径)`

### 术语翻译

保持以下术语翻译一致：
- Scout - 侦察员
- Peak - 山峰
- Shore - 海岸
- Flare - 信号弹
- Ascent - 天阶（难度）
- Biome - 生物群系
- Campfire - 篝火
- BingBong Airlines - BingBong航空公司
