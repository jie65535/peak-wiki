# 图片资源

本目录包含Wiki使用的所有图片资源。

## 目录结构

```
images/
├── items/              # 道具图标
│   ├── x512/          # 512x512 高分辨率图标
│   └── x64/           # 64x64 标准图标
└── other/             # 其他图片（截图、示意图等）
```

## 道具图标

游戏内所有道具都有两种规格的图标：

### x512 (512x512)
- 高分辨率版本
- 用于详细展示和特写
- 文件较大

### x64 (64x64)
- 标准尺寸版本
- 用于列表、表格等紧凑展示
- 文件较小，加载快

## 命名规范

图标文件应使用以下命名规范：
- 使用英文名称（与游戏内一致）
- 小写字母
- 单词之间用下划线连接
- 例如：`flare.png`, `first_aid_kit.png`

## 文件格式

- 推荐使用PNG格式（支持透明背景）
- JPG格式用于截图
- 保持适当的文件大小

## 添加图片

1. 将图片放入相应目录
2. 确保命名规范正确
3. 在文档中使用相对路径引用
4. 示例：`![信号弹](../images/items/x64/flare.png)`

---

*注意：上传图片前请确保您拥有使用权限，或图片来源于游戏本身。*
