# 生物群系 (Biomes)

PEAK的岛屿地图分为多个不同的生物群系（Biomes），每个都有独特的地形、气候条件、危险和可收集的物品。玩家需要从海岸出发，依次攀爬通过各个生物群系，最终到达顶峰呼叫救援。

## 概述

当前地图每24小时在 [17:00 UTC](https://dateful.com/convert/utc?t=17) 更换一次，生物群系的内部布局和危险会随机生成。

## 大厅区域

### 机场 (Airport)
- 游戏开始前的大厅区域
- 可以邀请好友一起开始攀登
- 使用护照 (Passport) 自定义童子军外观
- 等待开始时可以玩棋类游戏或使用照相亭等娱乐设施
- 可在机场终端设置天阶 (Ascents) 难度等级

## 生物群系列表

### 区域 1：海岸 (Shore)

**地形特点**：
- 攀爬岩石峭壁
- 穿越木质桥梁（承重有限，太多童子军会压断）

**可获取食物**：
- 在棕榈树上找到椰子 (Coconuts)
- 在灌木丛中找到脆莓 (Crispberries)

**主要危险**：
- 🟢 **有毒危险**：水母、巨型海胆
- 木桥承重限制

**相关成就**：
- **赶海人奖章** (BEACHCOMBER BADGE)：爬过海岸

---

### 区域 2：雨林 (Tropics)

**地形特点**：
- 攀爬树木、树根和藤蔓
- 热带雨林环境

**可获取食物**：
- 各种自然食物：荔莓 (Kingberries)、火烧莓 (Scorchberries)、莓蕉 (Berrynanas)
- 从蜂巢 (Beehives) 中偷取蜂巢蜜 (Honeycombs)（小心被蜜蜂蜇！）

**主要危险**：
- 🟢 **有毒危险**：大型树根上的尖刺、爆炸蘑菇、蜱虫 (Ticks)
- 蜜蜂攻击

**相关成就**：
- **辟路者奖章** (TRAILBLAZER BADGE)：爬过雨林

---

### 区域 2（替代）：森蕈 (Roots)

**地形特点**：
- 在沼泽中攀爬高大的树木和巨型蘑菇
- 昏暗的红木森林环境

**可获取食物和道具**：
- 采集各种真菌，既可食用也可用作攀登工具
- 各类蘑菇

**主要危险**：
- 孢子
- 各种敌对昆虫
- 真菌僵尸 (Zombies)

**游戏机制**：
- 这是替代生物群系，在某些地图中会替代雨林出现
- 2024年11月17日后，未来地图会在雨林和森蕈之间随机轮换

---

### 区域 3：雪山 (Alpine)

**地形特点**：
- 穿越冰冻的寒风和冰洞
- 极寒环境

**可获取食物**：
- 不同种类的雪莓 (Winterberries)
- 葚莓 (Clusterberries)

**主要危险**：
- 🔵 **寒冷危险**：需要躲避冰冻暴风雪
- 冰岩 (Ice Rocks)
- 🔴 **热量危险**：喷发的间歇泉 (Geysers) 会灼伤并将你击飞

**生存要点**：
> "当我们接近顶峰，气温也就开始急剧下降。持续暴露在这种强风之中可能会害死性命，一定要找到稳定的遮蔽处才行。"
> —— 童军领队迈尔斯

**相关成就**：
- **登山家奖章** (ALPINIST BADGE)：爬过雪山
- **透心凉奖章**：在全程寒冷值未超过 20% 的情况下爬过雪山

---

### 区域 3（替代）：方山 (Mesa)

**地形特点**：
- 穿越炙热阳光下的沙质地形

**主要危险**：
- 🔴 **炙热危险**：需要躲避烈日
- 🟤 **荆棘危险**：仙人掌 (Cacti)、滚动的风滚草 (Tumbleweeds)
- 🟢 **有毒危险**：毒蝎 (Scorpions)
- 💥 **爆炸危险**：炸药 (Dynamite)
- 其他危险

**游戏机制**：
- 这是替代生物群系，在某些地图中会替代雪山出现
- 添加于更新1.20.a后，未来地图会在方山和雪山之间随机轮换

---

### 区域 4：火山 (Caldera)

**地形特点**：
- 在燃烧的岩石间跳跃
- 需要等待熔岩涨落再跳跃

**可获取食物**：
- 在鸟巢中找到大蛋 (Big Eggs)
- 刺莓 (Prickleberries)

**主要危险**：
- 🔴 **热量危险**：避免站在燃烧的岩石上
- 上升和下降的熔岩

**相关成就**：
- **火山学奖章** (VOLCANOLOGY BADGE)：爬过火山

---

### 区域 5：熔炉 (The Kiln)

**地形特点**：
- 狭窄的垂直攀登路径
- 需要小心谨慎地向上攀登

**攀登策略**：
- 使用装备使攀登到顶峰 (Peak) 更容易
- 避开热岩 (Hot Rocks) 和上升的熔岩

**特殊说明**：
- 这是通往顶峰前的最后关卡
- 地形狭窄，需要精准的操作

---

### 区域 6：顶峰 (Peak)

**到达目标**：
- 🎉 **你成功抵达了顶峰！恭喜！**

**关键道具**：
- 可以在这里找到一箱信号弹 (Flares) 用来呼叫救援
- ⚠️ **注意**：在天阶4或更高难度下，顶峰没有信号弹

**相关成就**：
- **顶峰奖章** (PEAK BADGE)：抵达顶峰

**童军领队的日记**：
> "首要之事：要想被找到，就往高处跑。光在神秘岛屿的海滩上胡乱闲逛可不会帮你获救。所以，找到最近的山峰，然后向那里进发吧。"

## 营火系统 (Campfires)

每个生物群系之间都有营火站，童子军可以在这里：

### 营火功能
- **烹饪食物**：将生食烹饪成熟食，提升效果并去除负面状态
- **恢复体力**：在营火旁休息可以恢复体力
- **提升士气**：获得士气提升效果
- **储存物品**：暂时放置物品
- **队友会合点**：与队友集结的重要地点

### 相关成就
- **烹饪奖章**：在营火处做 20 次饭
- **欢乐露营奖章**：从营火处获得 5 次士气提升

## 每日地图机制

### 地图更新规则
- **更新时间**：每天 17:00 UTC
- **生物群系顺序**：固定（海岸→雨林/森蕈→雪山/方山→火山→熔炉→顶峰）
- **替代生物群系**：雨林/森蕈、雪山/方山会随机轮换出现
- **内部布局**：每次更新时随机生成
- **自然食物**：每次探险随机生成
- **行李箱内容**：每次探险随机生成

### 环境危险
每个生物群系都有其独特的环境危险：
- **有毒** 🟢：中毒值累积
- **寒冷** 🔵：体温下降
- **热量** 🔴：过热灼伤
- **荆棘** 🟤：尖刺伤害
- **爆炸** 💥：爆炸伤害

## 攀登策略

### 初学者提示
1. **在每个营火站充分休整**：恢复体力和士气，烹饪食物
2. **收集食物和装备**：在路上尽量收集资源
3. **避开危险区域**：不要冒险进入危险地带
4. **与队友保持联系**：多人模式下保持团队协作
5. **携带必需品**：至少携带一些食物和医疗用品

### 高级技巧
1. **规划最优路线**：研究地图布局，选择最短最安全的路线
2. **合理分配资源**：根据生物群系特点准备相应物品
3. **掌握环境特点**：利用每个生物群系的特性
4. **高效通过难点**：使用装备克服困难地形
5. **速通策略**：熟练后可以尝试一小时内登顶（匆忙奖章）

### 各生物群系准备建议

**海岸 (Shore)**：
- 准备解毒剂应对有毒危险
- 注意木桥承重，分批通过
- 收集脆莓作为初期食物

**雨林 (Tropics)**：
- 小心蜜蜂，快速取蜂巢蜜
- 准备解毒剂应对尖刺和蜱虫
- 收集各类浆果

**森蕈 (Roots)**：
- 大量采集蘑菇作为食物和工具
- 小心真菌僵尸
- 注意孢子危险

**雪山 (Alpine)**：
- 携带暖宝宝 (Heat Pack) 或火把 (Torch) 保暖
- 在暴风雪时躲避
- 注意间歇泉的喷发时机
- 收集雪莓和葚莓

**方山 (Mesa)**：
- 携带防晒喷雾 (Sunscreen) 或太阳伞 (Parasol)
- 躲避烈日
- 小心仙人掌和蝎子

**火山 (Caldera)**：
- 观察熔岩涨落节奏
- 快速通过燃烧岩石
- 不要在热岩上停留
- 收集大蛋和刺莓

**熔炉 (The Kiln)**：
- 使用绳索 (Rope Spool)、岩钉 (Piton) 等装备
- 谨慎攀爬，不要急躁
- 携带足够的体力恢复物品

**顶峰 (Peak)**：
- 使用信号弹 (Flare) 呼叫救援
- 天阶4+难度需自带信号弹

## 常见问题 (FAQ)

**Q: 地图什么时候更新？**
A: 每天 17:00 UTC 地图会更新，生成新的布局。

**Q: 替代生物群系是什么意思？**
A: 某些区域有两个不同的生物群系可能出现（雨林/森蕈，雪山/方山），每次地图更新时随机选择其中之一。

**Q: 如何知道当前是哪个生物群系？**
A: 通过环境特征和危险类型可以识别，每个生物群系都有独特的视觉风格。

**Q: 营火在哪里？**
A: 每两个生物群系之间都有营火，是固定的休息点。

**Q: 为什么顶峰没有信号弹？**
A: 在天阶4或更高难度下，顶峰不会提供信号弹，玩家需要自己携带。

**Q: 最难的生物群系是哪个？**
A: 这取决于个人风格和装备，但一般认为熔炉和火山由于环境危险密集而较难。

## 总结

PEAK的生物群系系统为游戏提供了丰富的多样性和挑战性。从海岸的起点到顶峰的终点，每个生物群系都有其独特的美学风格、危险类型和资源分布。掌握各个生物群系的特点，合理准备装备和食物，善用营火系统，是成功登顶的关键。

无论是单人独狼还是多人合作，理解和适应每个生物群系的机制都将帮助你更高效地完成攀登，解锁更多成就！
