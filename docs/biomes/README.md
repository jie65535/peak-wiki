# 生物群系

PEAK游戏中岛屿的各个生物群系介绍。

## 概述

岛屿被划分为多个生物群系，每个生物群系由营火（Campfire）分隔。地形、生物群系和陆地危险每天在**UTC时间17:00**随机生成，这意味着每天的岛屿布局都会不同。

## 生物群系列表

岛屿从低到高依次包含以下生物群系：

### 1. 海岸 (Shore)
- **位置**：起始区域
- **描述**：BingBong航空公司的飞机坠机地点
- **特点**：相对安全的起始区域，适合收集初始资源
- **环境危险**：基础环境危险
- **相关成就**：赶海人奖章 - 爬过海岸

### 2. 雨林 (Tropics)
- **位置**：海岸之上
- **描述**：茂密的热带雨林区域
- **特点**：植被丰富，可能有更多食物资源
- **环境危险**：待补充
- **相关成就**：辟路者奖章 - 爬过雨林

### 3. 雪山 (Alpine)
- **位置**：雨林之上
- **描述**：寒冷的高山区域
- **特点**：气温下降，环境更加恶劣
- **环境危险**：待补充
- **相关成就**：登山家奖章 - 爬过雪山

### 4. 火山 (Caldera)
- **位置**：雪山之上
- **描述**：活火山区域
- **特点**：极端环境，高温和岩浆危险
- **环境危险**：待补充
- **相关成就**：火山学奖章 - 爬过火山

### 5. 顶峰 (Peak)
- **位置**：岛屿最高点
- **描述**：游戏的终点目标
- **目标**：使用信号弹召唤救援直升机
- **特点**：最具挑战性的区域
- **相关成就**：顶峰奖章 - 抵达顶峰

## 环境要素

### 营火 (Campfire)
- **作用**：分隔不同的生物群系
- **功能**：
  - 烹饪食物（20次可获得烹饪奖章）
  - 提供士气提升（5次可获得欢乐露营奖章）
  - 可能作为检查点

### 环境危险
每个生物群系都包含特定的环境危险，需要玩家小心应对。具体危险因生物群系而异。

### 随机生成
- **地形**：每日UTC时间17:00随机生成
- **危险位置**：每日随机
- **布局**：每天都不同，需要重新探索

## 探索建议

1. **海岸**：收集初始资源，熟悉操作
2. **雨林**：注意收集食物和道具
3. **雪山**：准备应对寒冷环境
4. **火山**：小心高温和岩浆
5. **顶峰**：确保携带信号弹

---

*提示：由于每日随机生成，建议玩家每次游戏都要探索和适应新的地形布局。合理利用营火进行补给和烹饪。*
