# 生物群系

PEAK游戏中岛屿的各个生物群系介绍。

## 概述

岛屿被划分为多个生物群系，每个生物群系由篝火（Campfire）分隔。地形、生物群系和陆地危险每天在UTC时间17:00随机生成，这意味着每天的岛屿布局都会不同。

## 主要生物群系

### 海岸 (Shore)
- **描述**：游戏开始的地方，BingBong航空公司的飞机坠毁在这里
- **特点**：相对安全的起始区域
- **危险**：基础环境危险

### 山峰 (Peak)
- **描述**：岛屿的最高点，游戏的终点
- **目标**：到达山峰并使用信号弹召唤救援直升机
- **特点**：最具挑战性的区域

## 环境要素

### 篝火 (Campfire)
- **作用**：分隔不同的生物群系
- **功能**：可能提供休息和补给点

### 随机生成
- **地形**：每日随机
- **危险**：每日随机
- **更新时间**：UTC时间17:00

---

*提示：由于每日随机生成，建议玩家每次游戏都要探索和适应新的地形布局。*
