# 道具系统

PEAK游戏中的所有道具信息。

## 道具分类

### 关键道具

#### 信号弹 (Flare)
- **类型**：关键道具
- **用途**：在顶峰使用信号弹可以召唤救援直升机，完成游戏目标
- **获取方式**：探险途中寻找或从行李箱中获得
- **稀有度**：待确认
- **说明**：这是游戏的核心道具，到达顶峰后使用它来完成逃生。必须携带至顶峰才能完成游戏。

### 攀爬辅助道具

#### 岩钉 (Piton)
- **类型**：攀爬辅助
- **用途**：放置在岩壁上作为额外的抓点，辅助攀爬困难路段
- **获取方式**：待补充
- **使用方法**：在攀爬时对准墙面放置
- **成就相关**：放置10根可获得攀岩奖章
- **注意**：使用岩钉会影响环保奖章的获得

### 食物类

#### 浆果 (Berry)
- **类型**：食物
- **特点**：岛屿上自然生成，有多种类型
- **效果**：恢复生命值或提供特殊效果
- **随机性**：每次探险位置和类型随机
- **成就相关**：单次探险食用5种不同浆果可获得觅食奖章

#### 烹饪食物
- **类型**：食物
- **制作**：在营火处烹饪原材料
- **效果**：通常比生食效果更好
- **成就相关**：在营火处做20次饭可获得烹饪奖章

### 医疗道具

#### 解毒道具
- **类型**：医疗
- **用途**：恢复中毒状态，恢复中毒值
- **获取方式**：行李箱或特定位置
- **成就相关**：累计恢复200点中毒值可获得毒理学奖章

### 神秘道具

#### 神秘物品 (Mystical Items)
- **类型**：特殊
- **特点**：稀有且效果特殊的道具
- **获取方式**：探索隐藏区域或特殊事件
- **成就相关**：获得一件可获得玄学奖章

### 行李箱物品

行李箱是游戏中重要的道具来源：
- **位置**：散落在岛屿各处
- **内容**：每次探险随机生成
- **稀有度**：物品有不同的稀有度等级
- **建议**：探索时留意行李箱位置

## 稀有度系统

游戏中的道具按稀有度分为不同等级：
- **普通**：常见道具
- **稀有**：较难获得的道具
- **史诗**：非常稀有的道具
- **传说**：极其罕见的道具

稀有度影响道具的效果和获取难度。

## 道具图标

本Wiki包含游戏内所有道具的图标：
- **x512规格**：高分辨率图标（512x512），位于 `images/items/x512/`

图标可用于识别道具和制作攻略。

## 道具使用技巧

### 信号弹
- 必须携带到顶峰
- 使用前确保到达正确位置
- 是完成游戏的唯一方式

### 岩钉
- 在困难路段使用
- 可以节省体力
- 但会影响某些成就

### 食物
- 及时补充生命值
- 尝试烹饪获得更好效果
- 收集多种浆果

### 解毒道具
- 应对有毒环境时携带
- 根据需求使用
- 不要浪费

## 道具管理

### 背包系统
- 道具有数量限制（待确认）
- 合理选择携带物品
- 丢弃不需要的道具

### 道具共享
- 多人模式下可以传递道具
- 团队合作分配资源
- 确保关键道具（如信号弹）安全
