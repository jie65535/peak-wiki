# 道具系统

PEAK游戏中的所有道具信息。

## 道具分类

### 重要道具

#### 信号弹 (Flare)
- **用途**：在山峰（Peak）使用信号弹可以召唤救援直升机，完成游戏目标
- **获取方式**：待补充
- **说明**：这是游戏的核心道具，到达山峰后使用它来完成逃生

### 食物
岛屿上自然生成的食物，每次探险时内容随机生成。

### 行李箱道具
散落在岛屿各处的行李箱，内容每次探险随机生成。
