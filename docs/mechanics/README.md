# 游戏机制

PEAK游戏的核心机制和系统说明。

## 核心玩法

### 攀登系统
游戏的核心是攀登——从坠机地点的海岸出发，向岛屿最高点的顶峰前进。玩家需要：
- 穿越多个生物群系（海岸→雨林→雪山→火山→顶峰）
- 收集资源和道具
- 克服环境危险
- 最终使用信号弹召唤救援

### 合作模式
- **单人模式**：独自作为童子军进行探险
- **多人模式**：最多4名玩家组成童子军小队，协力攀登
- **语音聊天**：游戏包含语音聊天功能，建议使用耳机和麦克风

## 基础操作

### 移动与攀爬
- **移动**：在地面和平台上行走
- **攀爬**：抓住可攀爬表面进行攀登
- **跳跃**：越过障碍
- **冲刺/跳跃**：在攀爬时快速移动
- **蹲伏**：通过低矮空间

### 互动与合作
- **互动**：使用道具、打开行李箱
- **伸出援手**：帮助其他童子军
- **标记位置**：为队友标记重要位置
- **表情**：与队友交流

### 道具操作
- **丢下/投掷**：按住按键丢弃或投掷物品
- **瞄准**：精确瞄准目标

## 难度系统

### 攀登 (Ascents)
游戏有多个难度等级，称为"攀登"（Ascents）：
- 每个等级都会增加负面效果
- 负面效果会叠加，使游戏越来越困难
- 高难度提供更大挑战和成就感
- 为所有玩家统一应用

## 随机生成系统

### 每日随机（UTC 17:00更新）
以下内容每天在**UTC时间17:00**重新生成：
- **岛屿地形**：山体结构和路径
- **生物群系布局**：各区域的分布
- **陆地危险位置**：环境危险的具体位置

这意味着每天的岛屿都是全新的，需要重新探索。

### 每次探险随机
以下内容在每次探险时随机生成：
- **自然生成的食物**：浆果等可采集食物
- **行李箱内容**：散落的行李箱中的道具
- **稀有度**：物品的稀有度等级

## 资源系统

### 食物
- 岛屿上自然生成食物（如浆果）
- 可以在营火处烹饪
- 不同食物有不同效果
- 每次探险随机生成

### 行李箱
- 散落在岛屿各处
- 内容每次探险随机
- 可能包含有用道具
- 需要探索才能发现

### 稀有度系统 (Rarity)
游戏中的道具和物品有不同的稀有度等级，影响其属性和效果。

## 营火系统

营火是游戏中的重要设施，位于各生物群系之间：

### 功能
- **烹饪**：可以烹饪食物（20次获得烹饪奖章）
- **士气提升**：提供休息和士气加成（5次获得欢乐露营奖章）
- **分隔点**：标记生物群系边界
- **检查点**：可能作为进度保存点

## 辅助道具

### 岩钉 (Piton)
- 可放置在岩壁上辅助攀爬
- 提供额外抓点
- 放置10根可获得攀岩奖章
- 使用岩钉会影响环保奖章

### 其他道具
- 解毒道具：恢复中毒值
- 神秘道具：特殊效果
- 详见道具系统页面

## 生存要素

### 掉落伤害
- 从高处坠落会受到伤害
- 避免掉落伤害可获得气球奖章
- 小心攀爬是关键

### 中毒
- 某些环境或生物可能造成中毒
- 使用道具恢复中毒值
- 累计恢复200点可获得毒理学奖章

### 复活系统
- 童子军倒下后可被队友复活
- 单次探险复活3名可获得救星奖章
- 合作模式的重要机制

## 救援系统

### 目标完成步骤
1. 从海岸的坠机地点开始
2. 穿越所有生物群系
3. 到达顶峰
4. 使用信号弹
5. 召唤救援直升机
6. 成功逃离岛屿

### 胜利条件
- 至少一名童子军到达顶峰并使用信号弹
- 其他童子军可以选择是否一起逃离
- 未逃离的童子军可获得参与奖章

## 游戏模式

### 在线模式
- 与好友组队
- 可以邀请好友加入
- 支持语音聊天

### 离线模式
- 单人游玩
- 挑战独狼奖章
- 体验完整游戏内容

---

*提示：理解游戏机制对于成功攀登至关重要！合理利用营火、岩钉等辅助设施，与队友密切配合。*
