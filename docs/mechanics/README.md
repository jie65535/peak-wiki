# 游戏机制

PEAK游戏的核心机制和系统说明。

## 核心玩法

### 攀登系统
游戏的核心是攀登——从坠机地点的海岸出发，向岛屿最高点的山峰前进。

### 合作模式
- **单人模式**：独自作为侦察员进行探险
- **多人模式**：最多4名玩家组成侦察小队，协力攀登

## 难度系统

### 攀登 (Ascents)
游戏有多个难度等级，称为"攀登"（Ascents）：
- 每个等级都会增加负面效果
- 负面效果会叠加，使游戏越来越困难
- 为所有玩家增加挑战性

## 随机生成系统

### 每日随机
以下内容每天在**UTC时间17:00**重新生成：
- 岛屿地形
- 生物群系布局
- 陆地危险位置

### 每次探险随机
以下内容在每次探险时随机生成：
- 自然生成的食物
- 散落的行李箱内容

### 稀有度系统 (Rarity)
游戏中的道具和物品有不同的稀有度等级。

## 环境系统

### 环境危险
每个生物群系都可能出现环境危险，需要玩家小心应对。

### 篝火系统
篝火用于分隔不同的生物群系，可能提供检查点功能。

## 救援系统

### 目标完成
1. 到达山峰（Peak）
2. 使用信号弹（Flare）
3. 召唤救援直升机
4. 成功逃离岛屿

---

*提示：理解游戏机制对于成功攀登至关重要！*
