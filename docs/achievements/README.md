# 成就系统（勋章）

PEAK游戏中的完整成就系统指南。勋章（Badges），也称为勋荣（Accolades）或成就（Achievements），是玩家完成特定动作后获得的奖励。

## 概述

成就系统特点：
- **显示位置**：获得的勋章会显示在童子军装备的绶带上
- **奖励物品**：每个勋章都有相应的奖励装饰品
- **查看方式**：可在暂停菜单的"勋荣"界面查看完整列表
- **难度适用**：除"不死族接触奖章"外，所有勋章都可在任何难度下获得

游戏内会显示你的当前绶带和成功探险次数。

## 进度类勋章

这些勋章通过推进游戏进度自然获得。

### 参与奖章 (Participation Badge)
- **解锁条件**：让一名好友逃离岛屿而不带上你
- **难度**：⭐
- **说明**：需要多人游戏，让队友先行逃离而你留在岛上
- **奖励装饰品**：基础绶带

### 赶海人奖章 (Beachcomber Badge)
- **解锁条件**：爬过海岸
- **难度**：⭐
- **说明**：通过海岸生物群系，到达第一个营火点
- **技巧**：所有存活玩家在点燃营火时获得，即使刚被复活也算
- **奖励装饰品**：草帽

### 辟路者奖章 (Trailblazer Badge)
- **解锁条件**：爬过雨林
- **难度**：⭐⭐
- **说明**：通过雨林生物群系，到达第二个营火点
- **技巧**：所有存活玩家在点燃营火时获得
- **奖励装饰品**：丛林帽

### 登山家奖章 (Alpinist Badge)
- **解锁条件**：爬过雪山
- **难度**：⭐⭐⭐
- **说明**：通过雪山生物群系，到达第三个营火点
- **技巧**：所有存活玩家在点燃营火时获得
- **奖励装饰品**：飞行员帽

### 火山学奖章 (Volcanology Badge)
- **解锁条件**：爬过火山
- **难度**：⭐⭐⭐⭐
- **说明**：通过火山生物群系，到达最后一个营火点
- **技巧**：所有存活玩家在点燃营火时获得
- **奖励装饰品**：防火服

### 顶峰奖章 (Peak Badge)
- **解锁条件**：抵达顶峰
- **难度**：⭐⭐⭐⭐⭐
- **说明**：成功到达岛屿最高点，使用照明棒呼叫救援
- **技巧**：使用照明棒后成功登上直升机
- **奖励装饰品**：登山装备

## 活动类勋章

通过执行特定活动获得的勋章。

### 烹饪奖章 (Cooking Badge)
- **解锁条件**：在营火处做20次饭
- **难度**：⭐⭐
- **技巧**：每次在营火处烹饪食物都会计数，充分利用每个营火
- **奖励装饰品**：厨师帽

### 欢乐露营奖章 (Happy Camper Badge)
- **解锁条件**：从营火处获得5次士气提升
- **难度**：⭐⭐
- **技巧**：在营火旁休息可以获得士气提升效果
- **奖励装饰品**：营地装备

### 攀岩奖章 (Bouldering Badge)
- **解锁条件**：放置10根岩钉
- **难度**：⭐⭐
- **技巧**：在困难的攀爬路段使用岩钉辅助
- **奖励装饰品**：攀岩手套

### 毒理学奖章 (Toxicology Badge)
- **解锁条件**：使用道具恢复总计200点中毒值
- **难度**：⭐⭐⭐
- **技巧**：收集解毒剂等道具，主动食用有毒食物后解毒可加速进度
- **奖励装饰品**：防毒面具

### 觅食奖章 (Foraging Badge)
- **解锁条件**：单次探险食用5种不同浆果
- **难度**：⭐⭐
- **技巧**：注意收集各种浆果，不同颜色的脆莓、荔莓、莓蕉等都算
- **提示**：当给其他玩家喂食时，喂食者也会获得进度
- **食用第5种浆果时立即授予**
- **奖励装饰品**：采集篮

### 自然主义奖章 (Naturalist Badge)
- **解锁条件**：食用所有类型的自然食物
- **难度**：⭐⭐⭐
- **说明**：需要食用所有野生浆果、蘑菇等自然食物
- **技巧**：探索所有生物群系以找到稀有食物
- **奖励装饰品**：自然学家装备

### 美食家奖章 (Gourmand Badge)
- **解锁条件**：品尝所有类型的食物
- **难度**：⭐⭐⭐⭐
- **说明**：包括所有自然食物和包装食品
- **技巧**：需要多次游戏积累，从行李箱中寻找稀有食物
- **奖励装饰品**：美食家围裙

### 真菌学奖章 (Mycology Badge)
- **解锁条件**：食用所有类型的蘑菇
- **难度**：⭐⭐⭐
- **技巧**：探索森蕈生物群系，寻找各种蘑菇
- **奖励装饰品**：蘑菇帽

### 高等真菌学奖章 (Advanced Mycology Badge)
- **解锁条件**：单次探险食用所有5种菇莓
- **难度**：⭐⭐⭐
- **说明**：需要在一次探险中食用红、黄、绿、蓝、紫5种菇莓
- **技巧**：当给其他玩家喂食时，喂食者也会获得进度
- **食用第5种菇莓时立即授予**
- **奖励装饰品**：鹿角

### 急救奖章 (First Aid Badge)
- **解锁条件**：使用绷带或急救箱治疗累计500点受伤值
- **难度**：⭐⭐⭐
- **技巧**：收集医疗用品，帮助受伤的队友
- **奖励装饰品**：医疗包

## 挑战类勋章

需要特定技巧或策略才能完成的困难勋章。

### 独狼奖章 (Lone Wolf Badge)
- **解锁条件**：在单人探险中逃离岛屿
- **难度**：⭐⭐⭐⭐
- **技巧**：选择离线游玩或创建私人房间单人完成全程
- **奖励装饰品**：狼面具

### 救星奖章 (Clutch Badge)
- **解锁条件**：单次探险复活3名童子军
- **难度**：⭐⭐⭐
- **技巧**：需要多人游戏，及时救助倒下的队友
- **说明**：必须是你亲自复活3名队友
- **奖励装饰品**：英雄披风

### 气球奖章 (Balloon Badge)
- **解锁条件**：逃离岛屿且未遭受掉落伤害
- **难度**：⭐⭐⭐⭐
- **技巧**：小心攀爬，避免跌落，善用气球等道具缓冲
- **说明**：授予所有未受跌落伤害的存活玩家，即使中途被复活也算
- **奖励装饰品**：罗素的服装

### 环保奖章 (Leave No Trace Badge)
- **解锁条件**：逃离岛屿且未在山上放置任何东西
- **难度**：⭐⭐⭐⭐
- **技巧**：不使用岩钉、绳索等放置型道具完成攀登
- **说明**：完全依靠攀爬技巧，不留下任何物品
- **奖励装饰品**：环保徽章

### 匆忙奖章 (Hasty Badge)
- **解锁条件**：在一小时内逃离岛屿
- **难度**：⭐⭐⭐⭐
- **技巧**：熟悉路线，快速通过各个生物群系
- **说明**：需要高效的路线规划和团队协作
- **奖励装饰品**：速度靴

### 生存专家奖章 (Survivalist Badge)
- **解锁条件**：在第5级或更高攀登难度下逃离岛屿
- **难度**：⭐⭐⭐⭐⭐
- **技巧**：需要完善的策略和充足的准备
- **奖励装饰品：生存装备

### 坚韧奖章 (Endurance Badge)
- **解锁条件**：在单次探险中攀爬超过1000米
- **难度**：⭐⭐⭐⭐
- **技巧**：选择较难的攀登路线，延长攀爬距离
- **奖励装饰品**：耐力徽章

### 极限奖章 (Ultimate Badge)
- **解锁条件**：在最高难度下逃离岛屿
- **难度**：⭐⭐⭐⭐⭐
- **技巧**：需要完美的团队配合和策略
- **奖励装饰品**：传奇装备

## 特殊勋章

需要特定条件或隐藏要素才能解锁的勋章。

### 玄学奖章 (Esoterica Badge)
- **解锁条件**：获得一件神秘道具
- **难度**：⭐⭐⭐
- **技巧**：探索隐藏区域，寻找稀有的神秘道具
- **奖励装饰品**：神秘徽章

### 宾邦奖章 (Bing Bong Badge)
- **解锁条件**：携带宾邦逃离岛屿
- **难度**：⭐⭐⭐
- **说明**：只要有一名存活玩家携带宾邦，所有玩家（包括死亡玩家）都会获得此奖章
- **奖励装饰品**：螺旋桨帽

### 航天奖章 (Aeronautics Badge)
- **解锁条件**：实现飞行
- **难度**：⭐⭐⭐
- **说明**：需要总共6个气球（单个气球或一束气球）
- **技巧**：装备所有6个气球并漂浮即可获得
- **奖励装饰品**：宇航员头盔和服装

### 24K奖章 (24 Karat Badge)
- **解锁条件**：向窑炉献上贵重祭品
- **难度**：⭐⭐⭐⭐
- **步骤**（包含剧透）：
  1. 在台地中找到通往古墓的入口
  2. 通过跑酷谜题，收集终点的古老玩偶
  3. 将玩偶带到窑炉处扔进岩浆中
  4. 当玩偶被摧毁时，所有存活玩家都会获得此奖章，无论是谁扔的
- **奖励装饰品**：宾邦小帽

### 对牛弹琴奖章 (Animal Serenading Badge)

- **说明**：水豚可在雪山和台地找到

- **奖励装饰品**：螃蟹帽

### 树艺师奖章 (Arborist Badge)

## 成就策略

### 快速解锁建议

**新手优先勋章**：
1. 赶海人奖章 - 完成第一个生物群系
2. 烹饪奖章 - 在营火处多次烹饪
3. 觅食奖章 - 收集5种不同浆果
4. 攀岩奖章 - 使用10根岩钉

**中级目标勋章**：
1. 辟路者奖章 - 到达雨林
2. 登山家奖章 - 到达雪山
3. 毒理学奖章 - 累计解毒200点
4. 对牛弹琴奖章 - 为水豚演奏

**高级挑战勋章**：
1. 火山学奖章 - 到达火山
2. 顶峰奖章 - 成功登顶
3. 独狼奖章 - 单人完成
4. 气球奖章 - 无跌落伤害完成

### 成就组合策略

**单次探险多成就**：
- 觅食奖章 + 烹饪奖章：收集多种浆果并在营火烹饪
- 攀岩奖章 + 环保奖章的对立：二选一规划
- 救星奖章 + 急救奖章：在多人游戏中积极救援

**需要多次探险**：
- 美食家奖章：需要品尝所有食物类型
- 自然主义奖章：需要食用所有自然食物
- 盗贼奖章：需要打开大量行李箱

### 困难勋章攻略

**独狼奖章**：
- 选择较低难度开始
- 熟悉每个生物群系的路线
- 准备充足的食物和医疗用品
- 谨慎攀爬，避免不必要的风险

**气球奖章**：
- 收集气球等缓冲道具
- 避免冒险跳跃
- 使用岩钉和绳索提高安全性
- 团队协作，互相照应

**环保奖章**：
- 依靠纯粹的攀爬技巧
- 熟练掌握跳跃和抓握时机
- 选择自然路径而非捷径
- 可能需要多次尝试熟悉路线

**极限奖章**：
- 组建经验丰富的团队
- 完善的策略和物资分配
- 保持团队通讯和协作
- 做好多次尝试的准备

## 常见问题

**Q: 哪些勋章最容易获得？**
A: 进度类勋章（赶海人、辟路者等）和基础活动勋章（烹饪、觅食）最容易获得。

**Q: 独狼奖章需要什么难度？**
A: 可以在任何难度（包括入门）下获得，但建议先在较低难度熟悉路线。

**Q: 气球奖章是否允许复活？**
A: 是的，即使中途被复活，只要最终逃离时未受过跌落伤害即可获得。

**Q: 多人游戏中谁能获得勋章？**
A: 大多数勋章授予满足条件的所有存活玩家。部分勋章（如救星、对牛弹琴）只授予执行动作的玩家。

**Q: 不死族接触奖章为何特殊？**
A: 这是唯一无法在入门难度获得的勋章，需要在更高难度下使用骨之书复活队友。

**Q: 如何查看已获得的勋章？**
A: 在暂停菜单中选择"勋荣"查看完整列表。未解锁的勋章会显示为问号。

**Q: 成就进度会保存吗？**
A: 大部分累计型成就（如毒理学、烹饪）的进度会跨探险保存。单次探险型成就必须在一次探险中完成。

**Q: 队友给我喂食算不算觅食进度？**
A: 是的，当其他玩家给你喂食浆果时，喂食者和被喂食者都会获得觅食进度。
