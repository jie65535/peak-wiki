# 成就系统

PEAK游戏中的勋章成就系统。

## 概述

游戏中可以通过完成各种挑战来获得勋章（Badge）。这些成就记录了你的攀登历程和游戏成就。

## 统计数据

- **已登上顶峰**：记录你成功登顶的次数

## 进度类勋章

### 参与奖章 (Participation Badge)
- **解锁条件**：让一名好友逃离岛屿而不带上你
- **难度**：⭐
- **说明**：需要多人游戏，让队友先行逃离

### 赶海人奖章 (Beachcomber Badge)
- **解锁条件**：爬过海岸
- **难度**：⭐
- **说明**：到达第一个生物群系之外

### 辟路者奖章 (Trailblazer Badge)
- **解锁条件**：爬过雨林
- **难度**：⭐⭐
- **说明**：到达第二个生物群系之外

### 登山家奖章 (Alpinist Badge)
- **解锁条件**：爬过雪山
- **难度**：⭐⭐⭐
- **说明**：到达第三个生物群系之外

### 火山学奖章 (Volcanology Badge)
- **解锁条件**：爬过火山
- **难度**：⭐⭐⭐⭐
- **说明**：到达最后一个生物群系之外

### 顶峰奖章 (Peak Badge)
- **解锁条件**：抵达顶峰
- **难度**：⭐⭐⭐⭐⭐
- **说明**：成功到达岛屿最高点

## 活动类勋章

### 烹饪奖章 (Cooking Badge)
- **解锁条件**：在营火处做20次饭
- **难度**：⭐⭐
- **技巧**：充分利用每个营火进行烹饪

### 欢乐露营奖章 (Happy Camper Badge)
- **解锁条件**：从营火处获得5次士气提升
- **难度**：⭐⭐
- **技巧**：在营火处休息获得士气提升

### 攀岩奖章 (Bouldering Badge)
- **解锁条件**：放置10根岩钉
- **难度**：⭐⭐
- **技巧**：使用岩钉辅助攀爬

### 毒理学奖章 (Toxicology Badge)
- **解锁条件**：使用道具恢复总计200点中毒值
- **难度**：⭐⭐⭐
- **技巧**：收集解毒道具，应对有毒环境

### 觅食奖章 (Foraging Badge)
- **解锁条件**：单次探险食用5种不同浆果
- **难度**：⭐⭐
- **技巧**：注意收集各种浆果

### 玄学奖章 (Esoterica Badge)
- **解锁条件**：获得一件神秘道具
- **难度**：⭐⭐⭐
- **技巧**：探索隐藏区域或特殊事件

## 挑战类勋章

### 独狼奖章 (Lone Wolf Badge)
- **解锁条件**：在单人探险中逃离岛屿
- **难度**：⭐⭐⭐⭐
- **技巧**：单人游戏完成全程攀登

### 救星奖章 (Clutch Badge)
- **解锁条件**：单次探险复活3名童子军
- **难度**：⭐⭐⭐
- **技巧**：需要多人游戏，及时救助倒下的队友

### 气球奖章 (Balloon Badge)
- **解锁条件**：逃离岛屿且未遭受掉落伤害
- **难度**：⭐⭐⭐⭐
- **技巧**：小心攀爬，避免跌落

### 环保奖章 (Leave No Trace Badge)
- **解锁条件**：逃离岛屿且未在山上放置任何东西
- **难度**：⭐⭐⭐⭐
- **技巧**：不使用岩钉等辅助道具完成攀登

### 匆忙奖章 (Hasty Badge)
- **解锁条件**：在一小时内逃离岛屿
- **难度**：⭐⭐⭐⭐⭐
- **技巧**：规划最优路线，快速攀登

## 成就攻略建议

### 新手推荐
1. 先获得进度类勋章，熟悉各个生物群系
2. 专注于烹饪和营火相关成就
3. 逐步挑战难度更高的成就

### 进阶挑战
1. 尝试单人完成（独狼奖章）
2. 挑战速通（匆忙奖章）
3. 完美攀登（气球奖章、环保奖章）

### 多人合作
1. 组队更容易获得救星奖章
2. 可以互相帮助完成困难成就
3. 参与奖章需要特殊配合

---

*提示：合理规划探险目标，可以在一次攀登中完成多个成就！*
