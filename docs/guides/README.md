# 游戏指南

PEAK游戏的新手指南和进阶攻略。

## 新手入门

### 游戏开始
1. 从BingBong航空公司飞机坠毁后醒来
2. 你身处岛屿的海岸（Shore）
3. 目标：到达山峰（Peak）并使用信号弹召唤救援

### 侦察员（Scout）
- 游戏中玩家扮演的角色
- 可单独行动或组队（最多4人）

## 生存技巧

### 资源管理
- 收集岛屿上自然生成的食物
- 搜索散落的行李箱获取道具
- 注意：食物和行李箱内容每次探险都不同

### 路线规划
- 岛屿地形每日UTC时间17:00更新
- 通过篝火在生物群系间移动
- 注意环境危险

## 多人合作

### 组队建议
- 最多可组成4人侦察小队
- 团队协作可以应对更高难度
- 互相帮助度过危险区域

## 难度选择

### 攀登等级
- 从较低的攀登等级开始熟悉游戏
- 逐步挑战更高难度
- 注意：高难度会叠加多个负面效果

## 常见问题 (FAQ)

### Q: 游戏目标是什么？
A: 从坠机地点出发，到达山峰（Peak），使用信号弹召唤救援直升机。

### Q: 地图是固定的吗？
A: 不是。地形、生物群系和陆地危险每天UTC时间17:00重新生成。

### Q: 可以和朋友一起玩吗？
A: 可以。游戏支持最多4人组队合作。

### Q: 什么是攀登（Ascents）？
A: 游戏的难度等级系统，每个等级都会增加额外的挑战。
