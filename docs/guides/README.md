# 游戏指南

PEAK游戏的新手指南和进阶攻略。

## 新手入门

### 游戏开始
1. 选择"创建游戏"或"离线游玩"
2. 从BingBong航空公司飞机坠机后醒来
3. 你身处岛屿的海岸（Shore）
4. 目标：到达顶峰（Peak）并使用信号弹召唤救援

### 首次游玩建议
1. **选择较低难度**：从基础攀登开始
2. **熟悉操作**：练习移动、攀爬、跳跃
3. **探索海岸**：在安全区域熟悉环境
4. **收集资源**：寻找食物和行李箱
5. **不要着急**：慢慢适应游戏节奏

### 基本操作指南

#### 移动控制
- **移动**：WASD或摇杆
- **跳跃**：空格键或按钮
- **蹲伏**：Ctrl或按钮
- **冲刺**：Shift或按钮

#### 攀爬技巧
- **抓住墙面**：靠近可攀爬表面自动抓取
- **攀爬移动**：使用移动键在墙面上移动
- **跳跃**：攀爬时跳跃可快速移动
- **冲刺**：攀爬时冲刺可做长距离跳跃

#### 互动操作
- **互动**：E键或按钮（拾取道具、使用物品）
- **伸出援手**：帮助倒下的队友
- **标记位置**：为队友标记重要地点
- **表情**：与队友交流

## 生存技巧

### 资源管理
- **食物**：
  - 收集岛屿上的浆果等食物
  - 尝试在营火处烹饪
  - 不同食物可能有不同效果
  - 单次探险吃5种浆果可获觅食奖章
  
- **行李箱**：
  - 探索地图寻找散落的行李箱
  - 每次探险内容都不同
  - 可能包含有用道具

- **道具使用**：
  - 解毒道具：应对中毒状态
  - 岩钉：辅助攀爬困难路段
  - 神秘道具：探索发现特殊物品

### 路线规划
- **地形探索**：
  - 每天17:00 UTC地形重置
  - 每次游戏需要重新探索
  - 寻找最优攀登路线
  
- **营火利用**：
  - 在营火处烹饪和休息
  - 获得士气提升
  - 作为进度节点

- **避免危险**：
  - 注意环境危险
  - 小心掉落伤害
  - 留意有毒区域

### 攀登技巧
1. **观察地形**：寻找可攀爬的路径
2. **使用岩钉**：在困难路段放置岩钉
3. **控制跳跃**：避免不必要的坠落
4. **保存体力**：合理使用冲刺
5. **携带补给**：准备足够的食物和道具

## 多人合作

### 组队优势
- 可以互相复活
- 分担资源收集任务
- 一起应对困难路段
- 更容易完成某些成就

### 合作建议
- **语音沟通**：使用耳机和麦克风
- **标记位置**：为队友标记重要信息
- **互相帮助**：及时复活倒下的队友
- **分工合作**：有人探路，有人收集资源

### 邀请好友
1. 创建游戏
2. 选择"邀请好友"
3. 等待好友加入
4. 最多支持4人

## 成就攻略

### 简单成就
- **赶海人奖章**：爬过海岸
- **烹饪奖章**：营火处做20次饭
- **欢乐露营奖章**：获得5次士气提升

### 中等成就
- **辟路者奖章**：爬过雨林
- **登山家奖章**：爬过雪山
- **觅食奖章**：单次探险吃5种浆果

### 困难成就
- **独狼奖章**：单人逃离岛屿
- **气球奖章**：无掉落伤害完成
- **环保奖章**：不放置任何东西完成
- **匆忙奖章**：一小时内完成

## 常见问题 (FAQ)

### Q: 游戏目标是什么？
A: 从坠机地点出发，穿越所有生物群系到达顶峰，使用信号弹召唤救援直升机。

### Q: 地图是固定的吗？
A: 不是。地形、生物群系和陆地危险每天UTC时间17:00重新生成，每天都是新挑战。

### Q: 可以和朋友一起玩吗？
A: 可以。游戏支持最多4人组队合作，建议使用语音聊天。

### Q: 什么是天阶（Ascents）？
A: 游戏的难度等级系统，等级越高负面效果越多，挑战越大。

### Q: 如何获得成就？
A: 完成特定条件即可解锁勋章，如到达某个生物群系、完成特定次数的活动等。

### Q: 死亡后会怎样？
A: 多人模式下队友可以复活你；单人模式下探险失败，需要重新开始。

### Q: 营火有什么用？
A: 可以烹饪食物、获得士气提升，也是生物群系之间的分隔点。

### Q: 如何使用岩钉？
A: 在攀爬时对准墙面使用，可以放置岩钉作为抓点，放置10根获得攀岩奖章。

### Q: 每天17:00 UTC是几点？
A: UTC时间17:00，换算为北京时间（UTC+8）是次日凌晨1:00。

### Q: 信号弹在哪里？
A: 需要在探险过程中寻找或从行李箱中获得，是到达顶峰后召唤救援的关键道具。

## 进阶技巧

### 速通技巧
1. 熟记最优路线
2. 减少资源收集时间
3. 跳过非必要战斗
4. 充分利用冲刺和跳跃
5. 目标：一小时内完成（匆忙奖章）

### 无伤通关
1. 仔细规划每次跳跃
2. 不冒险走捷径
3. 充分利用岩钉
4. 目标：零掉落伤害（气球奖章）

### 极简挑战
1. 不使用岩钉
2. 不放置任何辅助道具
3. 纯粹依靠攀爬技术
4. 目标：什么都不放置（环保奖章）
